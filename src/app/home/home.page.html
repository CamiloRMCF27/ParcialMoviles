Registro de estudiantes
nuevos Bienvenido Descripción

Está aplicación ayudará a llevar un mejor registro de los nuevos estudiantes guardando fácilmente sus datos para poder revisarlos posteriormente y realizar modificaciones.


Instrucciones de uso

Para poder realizar uso de la aplicación primero el usuario se debe registrar, luego iniciará sesión con los datos registrados aquí se podrá loguear con el rol de estudiante o de administrador, el estudiante solo podrá visualizar información mientras que el administrador podrá acceder a los datos y realizar modificaciones en los registros.
